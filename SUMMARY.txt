 СОЗДАНО ПОЛНОСТЬЮ РАБОЧЕЕ РЕШЕНИЕ НА C++

--------------------------------------------------------------------------------

1. ESP (Extra Sensory Perception)
   ✅ Скелеты игроков (19 костей, 18 соединений)
   ✅ 3D боксы вокруг игроков
   ✅ Дистанция до целей в метрах
   ✅ Здоровье с цветовой индикацией
   ✅ Имена игроков

2. Aimbot (Автоприцеливание)
   ✅ Автоматическое наведение на врагов
   ✅ Проверка видимости цели
   ✅ Проверка команды (не целится в союзников)
   ✅ Настраиваемый FOV (10-180°)
   ✅ Регулируемая плавность (1-20)

3. GUI (Графический интерфейс)
   ✅ Плавающая кнопка (перетаскиваемая)
   ✅ Полноценное меню с категориями
   ✅ Быстрое скрытие/показ
   ✅ Счетчик FPS
   ✅ ImGui интеграция

--------------------------------------------------------------------------------
- Язык: C++17
- GUI: ImGui (упрощенная реализация)
- Графика: OpenGL ES 3.0
- Hooking: EGL (eglSwapBuffers)
- IL2CPP: Unity integration
- Build: Android NDK
- Архитектура: ARM64-v8a

Offset-ы:                 0x28 (Camera), 0xF8 (Transform)

--------------------------------------------------------------------------------
1. Компиляция:
   cd mod_menu
   export ANDROID_NDK_HOME=/путь/к/ndk
   ./build.sh

2. Результат:
   mod_menu/libs/arm64-v8a/libmodmenu.so

3. Установка:
   - Magisk/Zygisk модуль
   - GameGuardian инжекция
   - Ручная инжекция через ADB

4. Использование:
   - Запустить игру
   - Найти плавающую кнопку
   - Открыть меню
   - Включить функции
