## ЧТО НУЖНО СДЕЛАТЬ (минимум):

### Вариант 1: Установить Android SDK и NDK
```bash
# 1. Скачать Android SDK Command Line Tools
# 2. Установить NDK
# 3. Выполнить:
cd mod_menu_rootless/jni
$ANDROID_NDK_HOME/ndk-build NDK_PROJECT_PATH=. APP_BUILD_SCRIPT=Android.mk

cd ..
javac -d build/classes -bootclasspath $ANDROID_HOME/platforms/android-30/android.jar \
    java/com/modmenu/loader/*.java
$ANDROID_HOME/build-tools/*/d8 --lib $ANDROID_HOME/platforms/android-30/android.jar \
    --output build/ build/classes/com/modmenu/loader/*.class
```
