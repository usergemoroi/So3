# Standoff 2 Mod Menu

Полнофункциональный модуль на C++ для игры Standoff 2 с ESP (скелеты) и аимботом.

## Возможности

### ESP (Extra Sensory Perception)
- ✅ **Скелеты игроков** - отображение костей всех игроков
- ✅ **Боксы** - рамки вокруг игроков
- ✅ **Дистанция** - показывает расстояние до игроков
- ✅ **Здоровье** - отображает HP игроков
- ✅ **Имена** - показывает имена игроков

### Aimbot
- ✅ **Автоприцеливание** - автоматическая наводка на врагов
- ✅ **Проверка видимости** - работает только с видимыми целями
- ✅ **Проверка команды** - не целится в союзников
- ✅ **Настройка FOV** - регулируемый радиус захвата цели (10-180°)
- ✅ **Плавность** - настраиваемая плавность наведения (1-20)

### GUI
- ✅ **Плавающая кнопка** - можно перетаскивать
- ✅ **Меню настроек** - полноценное меню с настройками
- ✅ **Скрытие/Показ** - быстрое переключение видимости меню
- ✅ **Удобный интерфейс** - разделенные категории настроек

## Компиляция

### Требования
- Android NDK r21 или новее
- Linux/Windows/macOS

### Команды для компиляции

```bash
cd mod_menu/jni
ndk-build NDK_PROJECT_PATH=. APP_BUILD_SCRIPT=Android.mk
```

После компиляции библиотека будет находиться в:
```
mod_menu/libs/arm64-v8a/libmodmenu.so
```

## Установка

1. Скомпилируйте библиотеку (см. раздел "Компиляция")
2. Используйте любой инжектор для Android (например, Zygisk, Riru, или GameGuardian)
3. Инжектируйте `libmodmenu.so` в процесс игры

## Использование

### При запуске игры:
1. После инжекта в левом верхнем углу появится плавающая кнопка
2. Нажмите на кнопку чтобы открыть/закрыть меню
3. Включите нужные функции в меню

### Настройка ESP:
1. Откройте меню
2. Раздел "ESP Settings"
3. Включите "Enable ESP"
4. Выберите нужные опции (Skeleton, Box, Distance, Health, Name)

### Настройка Aimbot:
1. Откройте меню
2. Раздел "Aimbot Settings"
3. Включите "Enable Aimbot"
4. Настройте параметры:
   - **Visible Only** - цель только видимых врагов
   - **Team Check** - не целиться в союзников
   - **FOV** - угол захвата цели (чем меньше, тем точнее)
   - **Smooth** - плавность наведения (больше = плавнее)

## Технические детали

### Используемые технологии:
- **C++17** - современный стандарт C++
- **ImGui** - для создания GUI
- **OpenGL ES 3.0** - для рендеринга
- **IL2CPP** - для взаимодействия с Unity движком
- **Android NDK** - нативная разработка под Android

### Структуры из дампа:
- `PlayerController` - основной класс игрока (offset 0x28 - Transform)
- `Transform` - позиция и трансформация объектов
- `Vector3` - 3D координаты (x, y, z)
- `BipedMap` - карта костей персонажа (19 точек)

### Хуки:
- `eglSwapBuffers` - для рендеринга GUI и ESP
