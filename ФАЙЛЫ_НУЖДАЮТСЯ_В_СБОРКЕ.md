## Проблема:

В текущей среде разработки **НЕТ** Android SDK и NDK для фактической компиляции.

Для создания classes.dex нужен:
- android.jar (Android API classes)
- d8 инструмент (DEX компилятор)

Для создания libvis.so нужен:
- Android NDK с ndk-build
- ARM64/ARM32 toolchains

## Код готов - нужно только скомпилировать!

Все исходные файлы отредактированы и готовы к компиляции.
При наличии Android SDK и NDK, запуск следующей команды создаст файлы:

```bash
cd /home/engine/project/mod_menu_rootless/jni
$ANDROID_NDK_HOME/ndk-build NDK_PROJECT_PATH=. APP_BUILD_SCRIPT=Android.mk

cd /home/engine/project/mod_menu_rootless
mkdir -p build/classes
javac -d build/classes -source 1.8 -target 1.8 \
    -bootclasspath $ANDROID_HOME/platforms/android-30/android.jar \
    java/com/modmenu/loader/*.java
$ANDROID_HOME/build-tools/*/d8 --lib $ANDROID_HOME/platforms/android-30/android.jar \
    --output build/ $(find build/classes -name "*.class")
```
